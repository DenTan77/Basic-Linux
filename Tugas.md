1. Membuat folder proyek menggunakan Terminal Linux.
2. Membuat beberapa file (bisa tulisan, gambar, data atau yang lainnya).
3. Inisialisasi Git.
4. Melakukan beberapa commit.
5. Membuat branch baru.
6. Mengunggah repository ke GitHub.

Contoh pengerjaan:
```bash
mkdir proyek-latihan
cd proyek-latihan
mkdir docs
 touch README.md
 git init
 git add .
 git commit -m "Initial commit"
 git branch fitur-1
```

Contoh hasil yang diharapkan:
- Folder proyek berhasil dibuat.
- File README.md tersedia.
- Riwayat commit terlihat di Git.
- Branch baru berhasil dibuat.
